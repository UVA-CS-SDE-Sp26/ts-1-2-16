================
--- Purpose ---
================
Connects the FileHandler to the command line interface.
When executed with no arguments, it displays a numbered list of available files (Team Member B).
When file number provided, it retrieves the file contents and returns readable text (Team Member D's cipher).

============
--- UML ---
============
+String listFiles()
+String retrieve(int num)
+String retrieve(int num, String key)

==================
--- PseudoCode ---
==================
listFiles method:
    * get ArrayList of filenames from FileHandler
    * loop through list and format each file with a number
    * return all files combined into one string

retrieve(int num): default key
    * get filename at index num from FileHandler
    * get contents and pass to cipher decipher
    * return deciphered contents

retrieve(int num, String key):
    * get filename at index num
    * get file contents and pass contents WITH key to decipher
    * return deciphered contents
